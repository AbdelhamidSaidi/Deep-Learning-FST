# Deep-Learning-FST

A comprehensive collection of deep learning projects and tutorials covering various neural network architectures and applications.

## Repository Structure

### 📁 [consommation-lstm](consommation-lstm/)
LSTM-based models for fuel consumption prediction
- **fuel-lstm.ipynb** - LSTM model for predicting fuel consumption patterns

### 📁 [FashionMNIST](FashionMNIST/)
CNN and DenseNet implementations for fashion item classification
- **FashionMNIST-CNN-Densenet.ipynb** - CNN and DenseNet architectures
- **FashionMNIST-Improved.ipynb** - Enhanced model implementations
- **best_fashionmnist.pt** - Pre-trained model weights
- **data/** - FashionMNIST dataset (raw binary format)

### 📁 [LSTM](LSTM/)
LSTM tutorials and implementations
- **LSTM-I.ipynb** - LSTM fundamentals and implementations

### 📁 [transformers](transformers/)
Complete transformer model series covering NLP tasks
- **01_introduction.ipynb** - Introduction to transformers
- **02_classification.ipynb** - Text classification with transformers
- **03_transformer-anatomy.ipynb** - Deep dive into transformer architecture
- **04_multilingual-ner.ipynb** - Named Entity Recognition (multilingual)
- **05_text-generation.ipynb** - Text generation models
- **06_summarization.ipynb** - Text summarization
- **07_question_answering_v2.ipynb** - Question answering (v2)
- **08_model-compression.ipynb** - Model compression techniques
- **09_few-to-no-labels.ipynb** - Learning with limited labels
- **10_transformers-from-scratch.ipynb** - Building transformers from scratch
- **11_future-directions.ipynb** - Advanced topics and future directions
- **SageMaker/** - AWS SageMaker deployment examples
- **scripts/** - Utility scripts for training and processing
- **data/** - Dataset files (GitHub issues)
- **images/** - Reference visualizations

## Getting Started

Each folder contains Jupyter notebooks that can be run independently. All notebooks include:
- Data loading and preprocessing
- Model architecture definitions
- Training loops
- Evaluation metrics
- Visualization of results

## Requirements

See individual folder READMEs and `requirements.txt` files for specific dependencies.
